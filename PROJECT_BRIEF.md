# PROJECT_BRIEF.md

# 《烛之武退秦师》语言行动 RPG 研究 Demo：NPC Kernel 重制版

## 1. 项目定位

本项目是一个“语言行动驱动的角色扮演式社会模拟 RPG”研究 Demo。

玩家扮演烛之武，在秦晋围郑的历史局势中，通过自由自然语言与 NPC 互动。系统将玩家语言解析为结构化行动，并在世界状态机约束下推动 NPC 心智、势力关系、信息传播、事件演化和结局收束。

本项目不是：

```text
普通 AI NPC 聊天
传统 AVG 选项游戏
无限荒诞沙盒
全知旁白替所有人物编剧情
一个 LLM 统一揣测所有 NPC 的“上帝视角”故事生成器
```

本项目是：

```text
玩家语言行动
+ 客观世界状态机
+ 多个 NPC 独立心智内核
+ 事件可见性过滤
+ 硬约束验证
+ 组合式结局收束
```

## 2. 核心体验

玩家应该感到：

> 我不是在选择选项，也不是在和一个全知聊天机器人聊天；我是在扮演一个真实历史处境中的人，用语言、判断、情报和有限行动影响一个有多方主体的局势。

## 3. 设计哲学

### 3.1 语言是行动

玩家自然语言不是单纯聊天，而是角色在世界中的行动材料。

语言可以表达：

```text
说服
谈判
试探
交换利益
辩解
请求会面
调查
交付信息
等待
观察
```

系统必须先把语言解析成 `ActionProposal`，再让硬规则判断是否能成为世界事件。

### 3.2 自由度是“身份内自由”

玩家扮演烛之武，不是现代穿越者，不是全能上帝，不是荒诞玩家替身。

烛之武的身份边界：

```text
郑国老臣
忠于郑国
临危受命
熟悉外交与天下形势
无军权
无现代知识或现代武器
不会无故背叛郑国
不会做明显荒诞、违背身份的行为
```

允许玩家自由发挥的是：

```text
怎么劝
怎么试探
怎么交换利益
怎么争取会面
怎么处理秦晋裂痕
怎么利用郑国价值
怎么承担失败后果
```

不允许自由发挥的是：

```text
掏出现代武器
命令系统改状态
无因背叛郑国
跳出世界观操控秦伯
让 NPC 知道他不可能知道的事
```

### 3.3 关键 NPC 拥有独立心智内核

这是本版最重要的原则。

关键 NPC 不共享一个全知大脑。每个关键 NPC 都有自己的：

```text
身份
目标
性格
私有记忆
主观信念
误解
疑虑
情绪倾向
对玩家态度
对势力态度
可采取行动
已观察事件
```

同一个客观事件，对不同 NPC 会有不同可见版本。

例如：

```text
事件：烛之武在秦伯主帐中提出“亡郑利晋”。

秦伯看到：完整论证内容。
守卫看到：郑使与秦伯谈话时间变长。
晋使看到：郑使进入主帐且久未出来，但不知道具体话语。
郑伯看到：暂时什么都不知道，除非有人传信。
百姓看到：什么都不知道。
```

### 3.4 World Director 是调度器，不是总脑

World Director 不能替所有 NPC 思考，不能直接编排所有人物行为。

它只负责：

```text
判断哪些 NPC 应被某事件唤醒
把客观事件转换为每个 NPC 的 ObservableEvent
调度 NPC Kernel
收集 NPC 的行动提案
处理事件优先级
推进时间
触发结局裁决
```

它不负责：

```text
替秦伯思考
替晋使思考
替郑伯后悔
直接决定秦军撤退
直接写死剧情结果
```

### 3.5 LLM 只提案，不落盘

LLM 可以负责：

```text
理解玩家输入
生成 ActionProposal
在单个 NPC 私有上下文中推演主观反应
生成 NPC 台词
生成文学化叙事
辅助 World Director 总结局势
```

LLM 不允许直接修改 `WorldState`。

所有事实变化必须经过：

```text
ActionProposal / NPCIntention
→ Safety
→ Validator
→ GameEvent
→ Reducer
→ WorldState
```

### 3.6 状态机是世界宪法

状态机维护客观事实和硬约束：

```text
时间阶段
地点与连接
人物位置
谁在场
玩家身份
信息来源与可信度
NPC 可见事件
NPC 私有记忆摘要
事件日志
检查点
世界 flags
结局维度
```

它负责防止：

```text
LLM 胡编
NPC 全知
人物性格漂移
提示词攻击改世界
玩家越界行为生效
不可解释的状态跳变
```

## 4. MVP 成功标准

第一版不追求好玩度，而追求系统链路成立。

成功标准：

```text
1. 玩家自由输入 6 轮不崩。
2. 每轮输入能解析为结构化行动或安全拒绝。
3. 提示词攻击不能撼动世界状态。
4. 越界现代行为不能进入世界事实。
5. 关键 NPC 只能知道自己可见的信息。
6. 秦伯、郑伯、晋使的记忆与信念彼此独立。
7. 状态变化能从事件日志解释。
8. 至少能产生 3 条合理路线结局。
```
